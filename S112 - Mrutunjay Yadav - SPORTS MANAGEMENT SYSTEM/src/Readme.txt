SPORTS MANAGEMENT SYSTEM

The main objective of the College Sports Management system is discipline of the planning, organizing and managing the sport tasks.
Our project aims at making the task of sport easy. Sport Management is entering the records of new Players and retrieving the details of player available in the college.  
In the project we can maintain the  sports member who issue & return the issued sports kits.

Installation:

Install java netbeans & run the file.

Requirements:

Java Net beans 

Hardware requirement :

Operating system: ANY OS
Hard disks: 40GB
RAM: 256 MB
